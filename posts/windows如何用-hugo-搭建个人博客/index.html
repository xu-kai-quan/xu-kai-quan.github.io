Windows如何用 hugo 搭建个人博客

December 22, 2020

在Windows中安装hugo

  1. 去 Hugo releases页面下载 hugo_xxx_Windows-64bit.zip

  2. 解压后把文件夹名改为 hugo ，在D盘新建 Software 文件夹，把 hugo 文件夹放到里面

  3. 把 D:\Software\hugo\ 加到环境变量 PATH 中

  4. 运行 hugo version 查看版本 （如果这一步不成功，那一定是2，3步没有弄好）

搭建博客

  1. 进入Hugo官网

  2. 从 Step 2 开始看，因为 Step 1 是安装 hugo ，不过文档给的安装方式是是 macOS 的，所以不用看。

  3. 先进入一个安全目录（存放你的博客），然后执行 Step 2里的 hugo new site quickstart，这里的quickstart 建议换成 你的github名字.github.io_generator (现在，假设我的github名字是XXX)

  4. 如果你听从了我的建议，那么接下来一个一个的运行下面的命令

cd XXX.github.io_generator

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

echo 'theme = "ananke"' >> config.toml

hugo new posts/开博大吉.md

  1. 我们现在打开这个文件，就可以开始写博客了 把 draft 的值改为 false 。true 的意思是 这个只是草稿，他不会被发布

  2. 执行这个命令 hugo server ，意思是开启一个本地服务器，进入 http://localhost:1313/ 就可以看到你的博客了

  3. 上面只是在本地可以看到你的博客了，如果想要在别的地方也能看到，就需要配置GitHub Pages了

配置GitHub Pages

  1. 在 XXX.github.io_generator 目录中，新建一个文件叫 .gitignore ，注意这个点不能省，不是我打错了。然后进入这个文件，在里面输入 /public/。

  2. 然后执行 cd public ，进入 public 目录中 在执行

git init

git add .

git commit -m "第一次部署"

如何创建第二篇博客

  1. 在 XXX.github.io_generator 目录（注意确保自己不在 public 目录）里运行 hugo new posts/第二篇博客.md
  2. 然后对文件进行编辑
  3. 运行 hugo -D，得到新的 public 目录
  4. 进入 public 目录，执行一下操作
git add . 注意有一个点
git commit -m update
git push -f 其中 -f 是强制上传的意思
  1. 要等待个几分钟，你的博客才会出现。
